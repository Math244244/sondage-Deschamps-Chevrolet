━━━  1.  COMMENT ÇA FONCTIONNE?  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━  2.  2 FAÇONS D'ENVOYER LE SONDAGE  ━━━━━━━━━━━━━━━━━━━━━━━━━

Choisissez la méthode qui vous convient le mieux.


   🚀  MÉTHODE A — ENVOI AUTOMATISÉ VIA VOTRE CRM ACTIVIX
       ★ RECOMMANDÉ ★

   Vous pouvez intégrer un modèle d'email/SMS directement dans
   Activix pour que le sondage soit envoyé AUTOMATIQUEMENT au
   client dès que vous changez le statut du véhicule à « Livré ».

   Comment faire :

       1.  Contactez le support Activix (votre représentant CRM).
       2.  Demandez-leur de programmer un email/SMS automatisé qui
           se déclenche lorsque le véhicule passe au statut « Livré ».
       3.  Fournissez-leur l'un des modèles de message ci-dessous
           (Section 3) à utiliser comme gabarit.
       4.  Une fois configuré : plus rien à faire de votre côté!
           Le sondage part tout seul après chaque livraison. 🎉

   💡 AVANTAGE : Aucun oubli, 100% des clients reçoivent le sondage,
      gain de temps énorme pour votre équipe.


   📋  MÉTHODE B — ENVOI MANUEL (copier-coller)

   Si vous préférez envoyer le sondage manuellement après chaque
   vente, copiez-collez l'un des modèles de la Section 3 dans
   votre Gmail, Outlook ou application SMS.

   Procédure rapide :

       ÉTAPE 1   Choisissez votre modèle dans la Section 3
                 (Email avec bouton, Email texte, ou SMS).
       ÉTAPE 2   CTRL + A  (sélectionner)
                 CTRL + C  (copier)
       ÉTAPE 3   Collez dans Gmail / Outlook / SMS
                 CTRL + V
       ÉTAPE 4   Ajoutez le destinataire et envoyez! ✅



━━━  3.  VOS MODÈLES DE MESSAGE  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

3 formats disponibles. Utilisez celui qui correspond à votre besoin
(manuel ou pour Activix).


   📧  MODÈLE 1 — EMAIL AVEC BOUTON STYLISÉ (HTML)
       ► Le plus professionnel

   Ouvrez le fichier "Template-Email-Sondage.html" dans votre
   navigateur, puis CTRL + A → CTRL + C → collez dans Gmail/Outlook.

   Objet à utiliser :

       Merci pour votre achat! Un petit sondage rapide? ⭐

   💡 ASTUCE GMAIL — Sauvegardez comme modèle réutilisable :
      Nouveau message ► 3 petits points (en bas à droite) ► Modèles
      ► Enregistrer le brouillon comme modèle.

   💡 ASTUCE OUTLOOK — Sauvegardez comme modèle :
      Fichier ► Enregistrer sous ► Type : "Modèle Outlook (*.oft)".

Comment ça fonctionne?

Après chaque vente, vous envoyez un email ou un texto à votre client. Le client clique sur le lien et répond à 3 questions rapides (30 secondes).

✅ Client SATISFAIT  (Excellent + Excellent + Oui)
→ Redirigé automatiquement vers Google pour un avis 5 étoiles!

❌ Client PAS 100% satisfait
→ Message de remerciement interne seulement.
→ AUCUNE redirection Google. AUCUN mauvais avis public.

Choisissez votre méthode

📧  OPTION 1 — Email avec bouton stylisé  (le plus professionnel)
📝  OPTION 2 — Email texte simple  (prêt à copier-coller)
📱  OPTION 3 — SMS / texto  (prêt à copier-coller)

📧 Option 1 — Email avec bouton stylisé

Recommandé pour Gmail et Outlook (rendu visuel professionnel).

ÉTAPE 1Double-cliquez sur le fichier "Template-Email-Sondage.html".
► Il s'ouvre dans votre navigateur.
ÉTAPE 2CTRL + A  (sélectionner tout)
ÉTAPE 3CTRL + C  (copier)
ÉTAPE 4Gmail / Outlook ► Nouveau message ► Cliquez dans le corps du message ► CTRL + V (coller)
ÉTAPE 5Ajoutez l'adresse email du client.
ÉTAPE 6Ajoutez l'objet ci-dessous, puis ENVOYEZ.
Objet :  Merci pour votre achat! Un petit sondage rapide? ⭐
💡 Astuce Gmail — Sauvegardez comme modèle réutilisable :
Nouveau message ► 3 petits points (en bas à droite) ► Modèles ► Enregistrer le brouillon comme modèle.
💡 Astuce Outlook — Sauvegardez comme modèle :
Fichier ► Enregistrer sous ► Type : "Modèle Outlook (*.oft)".

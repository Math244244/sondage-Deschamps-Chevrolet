1Comment ça fonctionne?

Après chaque vente, vous envoyez un email ou un texto à votre client. Le client clique sur le lien et répond à 3 questions rapides (30 secondes).

✅ Client SATISFAIT  (Excellent + Excellent + Oui)
→ Redirigé automatiquement vers Google pour laisser un avis 5 étoiles!

❌ Client PAS 100% satisfait
→ Message de remerciement interne seulement.
→ AUCUNE redirection Google. AUCUN mauvais avis public.

22 façons d'envoyer le sondage

Choisissez la méthode qui vous convient le mieux.

🚀 Méthode A — Envoi AUTOMATISÉ via votre CRM Activix RECOMMANDÉ

Vous pouvez intégrer un modèle d'email/SMS directement dans Activix pour que le sondage soit envoyé automatiquement au client dès que vous changez le statut du véhicule à « Livré ».

Comment faire :

  1. Contactez le support Activix (votre représentant CRM).
  2. Demandez-leur de programmer un email/SMS automatisé qui se déclenche lorsque le véhicule passe au statut « Livré ».
  3. Fournissez-leur l'un des modèles de message ci-dessous (Section 3) à utiliser comme gabarit.
  4. Une fois configuré : plus rien à faire de votre côté, le sondage part tout seul après chaque livraison! 🎉
💡 Avantage : Aucun oubli, 100% des clients reçoivent le sondage, gain de temps énorme pour votre équipe.

📋 Méthode B — Envoi MANUEL (copier-coller)

Si vous préférez envoyer le sondage manuellement après chaque vente, copiez-collez l'un des modèles de la Section 3 dans votre Gmail, Outlook ou application SMS.

Procédure rapide :

ÉTAPE 1Choisissez votre modèle dans la Section 3 (Email avec bouton, Email texte, ou SMS).
ÉTAPE 2CTRL + A pour sélectionner, CTRL + C pour copier.
ÉTAPE 3Collez dans Gmail / Outlook / SMS  CTRL + V.
ÉTAPE 4Ajoutez le destinataire et envoyez! ✅
